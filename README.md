Proyecto CRUD hecho con **Node.js, Express y MySQL**, que permite gestionar una lista de películas con interfaz web.

---

## 🚀 Funcionalidades

- 📄 Listar películas
- ➕ Agregar películas
- ✏️ Editar películas
- 🗑️ Eliminar películas
- 🖼️ Soporte para imágenes de portada
- 🎭 Manejo de género, año y director

---

## 🛠️ Tecnologías usadas

- Node.js
- Express
- MySQL
- HTML
- CSS
- JavaScript
